Principais formatações

Negrito / Destaque (strong)

Nesta frase temos um termo em negrito usando a tag B (não semântica).

Nesta frese,tenho um termo em destaque usando a tag STRONG (semãntica).

Itálico / Ênfase (em)

Nesta frase, temos um termo em italico usando a tag I (não semântica).

Nesta frase temos um termo em enfase usando a tag EM (semantica).

Texto marcado (mark)

Podemos criar também um texto marcadousando a tag MARK

E no outro paragrafo temos outro texto marcado no final.

Texto grande e pequeno(small)

Estamos criando um texto grande e um texto pequeno nesse paragrafo.

Texto deletado (del)

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas nao considerado.

Texto inserido (ins)

Podemos marcar um texto como inserido para dar uma enfase e indicar que ele foi adicionado depois.

Texto sobrescrito (sup)

Para inserir coisas do tipo x20+3

Texto subscrito (sub)

Para inserir coisas do tipo H20
